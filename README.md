# TOOLKIVT

A lightweight AutoHotkey v2 toolkit designed to improve gameplay with a clean overlay, customizable controls and quality-of-life features.

## Features

- Clean in-game overlay
- Custom Trigger Key
- Z Boost support
- Help Center (F1)
- About Window
- Pause / Resume
- Pixel Debug
- Lightweight and fast
